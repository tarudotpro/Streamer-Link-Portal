# 特徴仕様: ライブ配信自動判別

## 概要
ユーザーが登録した YouTube URL が現在「ライブ配信中」であるかどうかを自動的に判別し、公開ページにバッジを表示します。

## 判別ロジック (MVP 案)
現時点では API クォータ節約のため、以下の段階的なアプローチを取ります。

1. **アイテム追加時**: 
   - OGP 情報を取得する際、メタデータ（`og:type` や `og:video:tag`）からライブ配信かどうかを推定。
   - YouTube の場合、URL に `live` を含むか、スクレイピング結果に `isLive` フラグがあるかを確認。
2. **ページ表示時 (キャッシュ付き)**:
   - 公開ページ（`/[username]`）のビルド（ISR）時に、短時間（例: 5分）のキャッシュを持って判定。

## UI/UX 定義
- **バッジ**: 
  - 色: 赤 (`#FF0000`)
  - テキスト: `🔴 LIVE` (点滅アニメーション付き推奨)
  - 配置: サムネイルの右上またはタイトル横
- **ソート順**: 
  - ライブ配信中のアイテムはリストの最上部に固定。

## 今後の課題 (Phase 2)
- YouTube Data API を使用した、より正確かつリアルタイムな状態取得。
- PubSubHubbub (WebSub) を利用した、配信開始・終了のリアルタイム検知とキャッシュ破棄。
