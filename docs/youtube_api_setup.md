# YouTube Data API v3 キーの取得と設定手順

システムが YouTube チャンネルの情報を自動取得するために必要な API キーの取得手順です。

## 1. Google Cloud Console でプロジェクトを作成
1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスします。
2. 左上のプロジェクト選択ドロップダウンから「新しいプロジェクト」を作成（または既存のものを選択）します。

## 2. API を有効にする
1. サイドメニューの「API とサービス」 > 「ライブラリ」を選択します。
2. 検索バーで「YouTube Data API v3」を検索し、選択して「有効にする」をクリックします。

## 3. API キーの作成
1. サイドメニューの「API とサービス」 > 「認証情報」を選択します。
2. 「認証情報を作成」 > 「API キー」をクリックします。
3. 生成された API キーをコピーして控えておきます。

## 4. プロジェクトへの設定
1. プロジェクトのルートディレクトリにある `.env.local` ファイルを開きます。
2. 以下の行を追加し、コピーしたキーを貼り付けます。
   ```env
   YOUTUBE_API_KEY=取得したAPIキー
   ```

## 5. (推奨) セキュリティ設定（キーの制限）
1. 作成した API キーの編集（鉛筆マーク）をクリックします。
2. 「API の制限」で「キーを制限」を選択し、プルダウンから「YouTube Data API v3」を選んで保存します。
   - これにより、万が一キーが漏洩しても YouTube API 以外には使われないようになります。
