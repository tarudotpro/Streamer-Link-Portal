## トラブルシューティング

### 開発サーバーが起動しない
```bash
# Node プロセスを停止
Stop-Process -Name node -Force -ErrorAction SilentlyContinue
# 再起動
npm run dev
```

### 環境変数が読み込まれない
開発サーバーを再起動してください（`.env.local` の変更は再起動が必要）

### Supabase 接続エラー
1. `.env.local` の内容を確認
2. Supabase ダッシュボードでプロジェクトが稼働中か確認
3. API キーが正しいか確認（Settings > API）
